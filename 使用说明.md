# 视频工厂 · 使用说明（拿到这个包先看这里）

一句话：输入产品（图/名/一句话）→ 自动写带货脚本 → 拆故事板分镜 → 出 AI 视频 → 拼接成片 → 配音 → 可批量。越用越好（自进化）。

---

## A. 给"人"用（3 步即开即用）

1. **装环境**（一次）：
   - Node.js 18+（https://nodejs.org ）
   - ffmpeg（长视频拼接用）：Windows `winget install ffmpeg` 或 choco；mac `brew install ffmpeg`
2. **填 key**：把 `.env.example` 复制成 `.env`，至少填这两个——
   - `KUAIZI_API_KEY=`（视频出片，快子·丽帧平台领，kz- 开头）
   - `IMAGE_API_KEY=`（免描述自动写脚本 + 文生图，hfsy 网关，sk- 开头；不填就自己写描述）
3. **启动**：双击 `start-video-factory.bat`（Windows）→ 自动开网页 + 起服务。**那个黑色命令行窗口保持打开**（它是出片引擎，关了就停）。浏览器里：上传产品图或输入一句话 → 选时长(可到 60s)/条数 → 生成。
   - mac/Linux：终端 `node live-server.js`，浏览器开 `http://localhost:8088`。

> 首次没装依赖时，命令行会提示装 Node/ffmpeg。生成 30/50/60s 会自动分多段并拼接成一条。

## C. 关键限制（诚实）

- 视频生成**必须经本地服务**（`live-server.js` :8088）中转：丽帧接口的浏览器跨域白名单不含 `ApiKey` 头，纯网页直连会被拦；本地服务无此限制。
- 单条视频 ≤15s；30/50/60s 由本机 ffmpeg 自动分段+拼接（所以要装 ffmpeg）。
- 上传图片/视频目前只做预览；"图生视频/真人素材混剪"需先把素材传到丽帧资产接口或公网图床拿 URL（本地文件丽帧服务器抓不到）——后续可接。
- 你的 key 只存在本机 `.env`，不进网页、不外泄。

## D. 自进化（越用越好）

每次生成的隐式信号（完成/重试/停留/下载/复用…，匿名、不采内容）→ 提炼 → 更新本地参数（`core/evolution/SkillParams`）→ 下次更好。命令 `node bin/cli.js evolve` 跑一轮。详见 `MASTER/视频工厂-总纲.md` §4 与 `EVOLUTION-SYSTEM-DESIGN.md`。
