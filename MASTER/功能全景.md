# 功能全景（全部功能清单 · 不漏）

> 标注：✅真连可用（已实测）/ ⚙️可跑（引擎内逻辑完整）/ 🧩骨架（接口在、需配 Key 或补实现）。

## A. 真实出片链路（✅ 已实测，live-server.js）
- ✅ **丽帧 Seedance 2.0 文生视频**：create→轮询 status→拿 video_url 下载 mp4（已实测真出片，5/10/15s）。
- ✅ **免描述自动脚本**：prompt 空 + 产品名/产品图 → 网关 LLM(gpt-5.5) 自动写带货分镜脚本（看图用视觉消息）。
- ✅ **长视频自动分段 + ffmpeg 拼接**：30→2 段、50/60→4 段，每段≤15s 并行生成 → 本机 ffmpeg 归一化(720x1280/24fps)concat 成一条（已实测 30s）。
- ✅ **矩阵多条**：count=N 并发生成 N 条；进度按分镜/条返回。
- ✅ **网关图片**：gpt-image-2pro 文生图（已实测出图 1.8MB）。
- ✅ **网页 UI**：`web/public/index.html`（file:// 双击或 localhost:8088 均可，自动连本地服务）；自定义时长(5/10/15/30/50/60+自定义4-60)、条数(1/3/5/10/20+自定义)、平台、国家、配音、上传素材预览(文件夹图标)、多条结果同屏播放、下载全部、断连自动重连。
- ✅ **一键启动** `start-video-factory.bat`：开网页 + 起服务（常驻窗口）。

## B. 抽象层 core/（⚙️ 引擎，模型无关）
- ⚙️ `VideoModel`/`TTSProvider` 统一接口 + 能力画像（ModelCapabilities/TTSCapabilities）。
- ⚙️ `ModelRegistry`/`TTSRegistry`：能力驱动选型 + 回退（按 mode/分辨率/时长/预算选最优，失败降级）。
- ⚙️ `VoiceLockManager` @voice1 声纹锁：全片单一声纹，AI 画面静音，统一音轨。
- ⚙️ `CostEstimator`：任意"模型×时长×条数×超分"组合估价。
- ⚙️ `QualityGate`：120 门质检（内容/真实/多样/合规/闭环/技术）。
- ⚙️ `runtime`(重试/并发/限流/日志)、`resilience`(熔断/超时/退避)、`observability`(JSON日志/指标/reqId)、`errors`(类型化)、`Config`(校验+env)。
- ⚙️ `JobStore`/`FileJobStore`(持久任务)、`Storage`(产物抽象)、`Queue`(幂等队列)、`BatchExecutor`(批量并发)、`FFmpegAssembler`(拼接)。

## C. 长视频 core/longvideo/（⚙️）
- ⚙️ `LongVideoPlanner`：按模型上限分段 + 续帧一致 + 主体恒定 subjectAnchor + 每 N 段用产品图再锚定（防漂移）。
- ⚙️ `LongFormAssembler`：归一化分辨率/帧率/SAR + concat/xfade 转场；支持混合用户素材。
- ⚙️ `AudioVideoMuxer`：ffmpeg 把配音贴到视频（对齐/采样率归一/BGM 混音/-c:v copy 无损）。
- ⚙️ `LongVideoPipeline`：端到端长视频编排。

## D. 自进化 core/evolution/ + hq/（⚙️）
- ⚙️ `SkillParams`：可训练参数（角度权重/模型偏置/提示后缀/质检门），版本化。
- ⚙️ `Telemetry`：匿名 opt-in 隐式遥测（哈希 id、白名单、不采内容/产品名/脚本）。
- ⚙️ `QualitySignals`：隐式质量（成功率/重试/质检通过/拼接成功/重试合成，无需打分）。
- ⚙️ `EvolutionEngine`：bandit + 验证门控（候选须严格优于当前 no ties）+ 学习率限幅 + 负样本缓冲 + 回滚。
- ⚙️ `Reporter`/`VersionChecker`：匿名上报 / 拉 `/params/latest` 本地再门控才采纳。
- ⚙️ `hq/collector`(收 /telemetry、发 /params/latest、/admin) + `hq/aggregate`(按版本聚合隐式质量 + 晋升/回滚)。【私有，不开源】

## E. 动态工作流 workflows/（⚙️）
- ⚙️ `WorkflowBuilder` 按 模式/预算/素材/矩阵/平台 组装 14 个标准 Stage（Context/Stage/Workflow）。

## F. 适配器 adapters/
- ✅ `LocalTtsAdapter`（本地 TTS，零云成本，对接 tts-server，默认 MOSS-TTS-Nano）。
- 🧩 video：Kling/Runway/Seedance/Veo（能力画像在，配 Key 可真连；当前真实出片走 live-server 丽帧直连）。
- 🧩 tts：ElevenLabs/CosyVoice/火山/GptSoVITS（骨架，配 Key 可用）。
- 🧩 新增模型只写一个 Adapter，核心零改动（插件式）。

## G. 本地 TTS 服务 tts-server/（⚙️ Python）
- ⚙️ FastAPI 包 MOSS/Kokoro/Piper → 统一 `/tts`、`/clone`、`/health`（默认 :9881）；零云 API 成本。
